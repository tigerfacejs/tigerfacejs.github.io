tigerface.js

网站刚刚创建，文档看起来还比较凌乱，请持续关注，每天都会有进展

tigerface.js 是一个基于 JavaScript 语言的图形和动画开发框架，适用于开发各种交互性、图形化项目， 例如：片头、游戏、图表、图形设计器。 如果你第一次来这里，可以先看看这里的几个简单范例，或者阅读一下我的使用教程。如果你是老朋友了，请直接访问我的 GitHub 项目， 或者点击下载按钮，直接下载项目压缩包。

开始下载 快速入门

colorscope 万花板：圆和三角形的相互作用下绘制出对称的图形，三角形被圆角处理并转化成多个点，调色板提供了变化的色彩

* 点击任意位置重新绘制

scrawl 涂鸦：粉笔在黑板上涂鸦的效果，CatmullRom插值绘制出圆滑的曲线，纹理产生笔刷效果，每次落笔随机取色

* 随意乱画，单击右下白色方块清除

pieChart 动态饼图：分析数据，精准绘制饼图，开场动画带来动感，封装成 React 组件，数据驱动重绘

* 点击任意位置模拟数据刷新

gum 口香糖：通过几次贝塞尔曲线绘制和填充，形成拉长变细的效果，结合内置的拖拽功能和补间算法

* 停不下来的想去拽

merge 合并：两个任意多边形，在移动中碰撞，多个顶点融合成新的多边形，随着位置改变不停的变化

* 只用静静的看着

balance 平衡：白色方块互相排斥，之间的连线却弹力拉扯，像粘性液体般的动态效果，较力中找到平衡状态

* 点击中心的红色方块随机打乱位置

cow 动画：这里利用了一下从游戏里提取的资源，按帧播出动作。

* 只用静静的看着
